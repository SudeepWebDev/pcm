Interactive Pilot Carrier Modulation Demo

How to use: Move the sliders to change the carrier, pilot, and message frequencies.
The chart below will update in real time to show the effect on the modulated signal.
What do these controls mean?
  • Carrier Frequency: This sets the frequency of the main carrier wave. Increasing it makes the green carrier wave oscillate faster. In real systems, this is the frequency at which information is transmitted.
  • Pilot Frequency: This sets the frequency of the pilot tone (orange). The pilot is a reference signal sent along with the main signal. Changing this shows how a pilot can be placed at a different frequency for synchronization or channel estimation.
  • Message Frequency: This sets the frequency of the message (blue) that modulates the carrier. Increasing it makes the information content change more rapidly.
What do we learn?
  • How the carrier and pilot combine with the message to form the modulated signal (red).
  • How changing each parameter affects the overall waveform and the role of the pilot in aiding demodulation and synchronization.
  • How pilot carrier modulation helps in real-world communication systems for robust and reliable data transfer.
Numerical Example:
Suppose: Message frequency = 20 Hz, Carrier frequency = 1000 Hz, Pilot frequency = 100 Hz.
Transmitted signal:
s(t) = sin(2π·20t)·cos(2π·1000t) + 0.2·cos(2π·100t)
Interpretation: The message modulates the carrier, and a small pilot tone is added at 100 Hz for reference.
